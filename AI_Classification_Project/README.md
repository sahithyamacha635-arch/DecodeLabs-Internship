# Iris Flower Classification

This is my DecodeLabs internship project based on machine learning classification.

In this project, I used the K-Nearest Neighbors (KNN) algorithm to classify iris flowers using their features.

Dataset used:

* iris.csv

Concepts used:

* dataset loading
* train test split
* KNN classification
* prediction
* confusion matrix
* accuracy checking

Tools used:

* Python
* Pandas
* Scikit-learn

Accuracy achieved:
96.66%
